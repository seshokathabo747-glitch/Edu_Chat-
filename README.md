📚 EduChat – Smart Learning Chat Platform

EduChat is a modern web-based chat application designed to help students communicate, learn, and collaborate in real time. It provides a simple and interactive space where learners can ask questions, share ideas, and support each other academically.

🚀 Features

- 💬 Real-time messaging using Firebase
- 👤 Usernames (no complicated sign-up required)
- 🎨 Clean and colorful user interface
- 🏠 Home page with chat access
- ⚙️ Settings page for customization
- ℹ️ About page explaining the platform
- 🌐 Works directly in the browser (HTML, CSS, JavaScript)

🎯 Purpose

EduChat was created to make learning more interactive and accessible for students. Instead of studying alone, learners can connect, discuss school topics, and solve problems together.

🛠️ Built With

- HTML
- CSS
- JavaScript
- Firebase Realtime Database

🔗 How It Works

Messages are stored and synced in real-time using Firebase, allowing users to instantly see new messages without refreshing the page.

🌍 Vision

To build a powerful student community platform that supports education, collaboration, and innovation—starting with simple chat and growing into a full learning ecosystem.
